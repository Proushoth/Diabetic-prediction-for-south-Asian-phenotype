# Diabetic-prediction-for-Sri-Lankan-phenotype

Diabetic Risk Predictor: South Asian Phenotype 🇱🇰
A Machine Learning-based non-invasive screening tool designed specifically for the South Asian "Thin-Fat" Phenotype. This project moves beyond traditional BMI standards by utilizing Waist-to-Height Ratio (WHtR) and sedentary behavior patterns to predict Type 2 Diabetes risk in urban professionals.

📌 Overview
Traditional BMI metrics often fail to identify metabolic risk in South Asians who may have a "normal" weight but high visceral adiposity. This system implements a Random Forest Classifier to provide a non-invasive, pain-free alternative to traditional screening.

Key Features
Ethnicity-Specific: Optimized for the Sri Lankan/South Asian biological profile.
Non-Invasive: No blood samples required; uses anthropometric and lifestyle data.
Explainable AI: Highlights the importance of WHtR over BMI.
Interactive Dashboard: Built with Streamlit for real-time user testing.
